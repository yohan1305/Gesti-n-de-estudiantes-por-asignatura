# Gesti-n-de-estudiantes-por-asignatura

Estimado profesor Francis:

Por medio de este informe y el repositorio compartido en GitHub, presento la práctica titulada “Gestión de Estudiantes por Asignatura”, correspondiente a la asignatura de Estructura de Datos.
El proyecto fue desarrollado en C# aplicando los conceptos de herencia, clases derivadas y polimorfismo, siguiendo las instrucciones establecidas en la guía de la práctica. La solución incluye:

- Definición de las clases necesarias (Estudiante, EstudiantePresencial, EstudianteDistancia, Grupo, Asignatura, Docente, OperationResult).
- Implementación de funcionalidades para agregar estudiantes, registrar calificaciones, mostrar listados y calcular porcentajes de aprobación.
- Uso de estructuras en memoria (List<T>) para almacenar los datos, sin recurrir a bases de datos.
- Aplicación de buenas prácticas de encapsulamiento, reutilización de código y abstracción.
- Ejecución del programa con ejemplos completos que permiten visualizar claramente el funcionamiento de cada operación.
El repositorio en GitHub contiene el código fuente comentado, junto con las capturas de pantalla que evidencian el correcto funcionamiento del sistema.
Con esta entrega se demuestra la comprensión y aplicación de los principios de programación orientada a objetos en un escenario realista de gestión académica.
